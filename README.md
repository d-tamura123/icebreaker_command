# icebreaker_command - 流氷溶かし隊
A 3D indie game set in 20XX, where global warming melts Antarctic ice and threatens world shipping routes. You are a member of the international Icebreaker Command, navigating your vessel to clear drifting ice and protect NPC trade ships using dynamic ocean flow fields. Built with DXLib + custom FlowField-based navigation.

20XX年、地球温暖化の進行により南極の氷が溶けだし、世界中の航路は流氷による深刻な危険にさらされている。
プレイヤーは国際組織「Icebreaker Command」の一員として船を操り、海流のFlowFieldを活用しながら流氷を対処し、NPC交易船の航路安全を確保する。
DXLib と独自の FlowField ナビゲーションを用いて構築された 3D インディーゲーム。
